Backend  : Python 3.12.3, Django 5.2.8 REST API 
Frontend : Angular 20.3.6 - Typescript
Database : Microst SQL Server

Dependency for Angular:
1. node 22.12.0
2. npm 11.5.1

Features:
1. User Registration and User Login
2. Multi-Factor Authenticator
   Must install Google or Microsoft Authenticator on your Mobile Phone for testing
3. User Profile, Change Password, Enable/Disable MFA and Upload Profile Picture
4. Product Pagination, Product List, Catalogs and Search with wild cards
